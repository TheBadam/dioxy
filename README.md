# DIOXY
Measure multiple environmental characteristics based on DFRobot SEN0335 for Micropython.
This combines libraries for ENS160 and BME280 sensors.
Will include Raspberry Pico 2 W sensor broadcasting.

ENS160 based on [awsiuk/ENS160](https://github.com/awsiuk/ENS160)
BME280 history explained in the class.
